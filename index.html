MNIST Digit Classification with kNN

Classified MNIST handwritten digits using kNN and Decision Tree algorithms. Preprocessed and analyzed a 70,000-image dataset. PCA reduced dimensionality, retaining 80% variance with 44 components. kNN achieved 97.49% accuracy with PCA, outperforming Decision Tree’s 65.55%. Demonstrated PCA’s role in boosting kNN accuracy, with visualizations and code on GitHub.
