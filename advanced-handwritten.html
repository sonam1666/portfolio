Advanced Handwritten Digit Classification

To begin with

In the quest to decipher handwritten digits from the MNIST dataset, I embarked on an exploration of two powerful machine learning algorithms: k-Nearest Neighbors (KNN) and Decision Trees. This project aimed to not only classify digits with precision but also to uncover the strengths and weaknesses of these algorithms in the realm of image recognition.

Technologies used

Programming Language: RLibraries: caret, class, factoextra, tidyverse, GGally, caTools, rpart, rpart.plot, randomForestTechniques: Data Normalization, Principal Component Analysis (PCA), k-Nearest Neighbors (KNN), Decision Trees

Data Exploration and Preprocessing

The journey began with a deep dive into the MNIST dataset, a treasure trove of 70,000 images of handwritten digits. I meticulously explored pixel distributions and patterns, preparing the data for the next steps. By normalizing the pixel values, I ensured that every image was scaled between 0 and 1, setting the stage for accurate and effective modeling.Dimensionality Reduction:

Simplifying Data with PCA

To simplify the challenge, I turned to Principal Component Analysis (PCA), a technique that reduces the complexity of data while preserving essential features. This step was crucial in making the subsequent analysis more manageable and insightful.

Model Evaluation:

Using the "elbow" method, a significant drop in explained variance is observed after the second dimension. This suggests that retaining just the first two dimensions captures the most crucial variance in the data, optimizing the analysis while preserving essential information.Â 

Programming in R

mnist_analysis.R
# MNIST Digit Classification with kNN and Decision Trees
# Set working directory and load required libraries
setwd("C:/Users/chava/Documents/Machine Learning/MNIST-data/")
library("caret")
library("class")
library("factoextra")
library("tidyverse")
library("GGally")
library("caTools")
library("rpart")
library("rpart.plot")
library("randomForest")
# Load and preprocess MNIST data
mnist_train <- read.csv("mnist_train.csv")
mnist_test <- read.csv("mnist_test.csv")
# Normalize pixel values
train_pixels <- mnist_train[, -1] / 255
test_pixels <- mnist_test[, -1] / 255
# Apply PCA for dimensionality reduction
pca_result <- prcomp(train_pixels)
train_pca <- predict(pca_result, train_pixels)[, 1:44]
# Train kNN model
knn_pred <- knn(train_pca, test_pca, mnist_train$label, k = 3)
# Calculate accuracy
accuracy <- sum(knn_pred == mnist_test$label) / length(mnist_test$label)
cat("kNN Accuracy with PCA:", accuracy * 100, "%\n")
# Train Decision Tree model
tree_model <- rpart(label ~ ., data = data.frame(train_pca, label = mnist_train$label))
tree_pred <- predict(tree_model, data.frame(test_pca), type = "class")
# Calculate Decision Tree accuracy
tree_accuracy <- sum(tree_pred == mnist_test$label) / length(mnist_test$label)
cat("Decision Tree Accuracy with PCA:", tree_accuracy * 100, "%\n")
