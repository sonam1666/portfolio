Data Visualisation of Sustainable Development Goals Progress

The spark of my interest

The SDGs are a blueprint for a better world, and this inspired me to explore how different countries are advancing towards these goals.

As someone deeply invested in understanding the global challenges we face today, I have always been intrigued by the concept of Sustainable Development Goals (SDGs).

Diving into the data

the "Sustainable Development Data up to 2023" dataset serves as a powerful tool for tracking the annual progress made by the 193 UN Member States towards achieving the SDGs. This dataset encompasses a rich array of data, including country names, country codes, years,SDG index scores, and individual scores for each of the 17 SDGs.

Visualizing Global Sustainability Progress :

In my initial analysis, I plotted the correlation between a country's overall SDG Index Score and its dedication to Quality Education (Goal 4). This scatter plot, color-coded by country, reveals consistent trends and highlights how countries with higher SDG Index Scores also perform well in Quality Education.

To guide my project, I focused on a bar graph that spotlights key sustainability goals such as clean water, affordable energy, Quality Education, and innovation. This detailed insight into the UK's sustainability trend informed my interactive graph, which showcases the average scores for each SDG goal worldwide. By analyzing individual countries, I added depth and context to the global sustainability trends.

The final visualization serves a dual purpose: it visually presents the average scores for each SDG goal, allowing easy identification of thriving areas and those needing more attention. It also features an interactive element, providing additional details like the title, average score, and year for each SDG goal. 

Programming in R

I developed an interactive Shiny application featuring a dynamic map and two graphs to visualize pollution data effectively. The interactive map identifies geographical hotspots of pollution incidents, aiding in targeted interventions. Additionally, the graphs depict trends over time.

sustainable_development_analysis.R
# Sustainable Development Goals Analysis
# Load required libraries
library("readr")
library("ggplot2")
library("dplyr")
library("tidyr")
library("visdat")
library("plotly")
#-----------------------------------------------------------------------
# Read in the data
data <- read_csv("sdg_index_2000-2022.csv")
#-----------------------------------------------------------------------
# Visualize missing values in the data
vis_miss(data)
#-----------------------------------------------------------------------
# Write the data to a CSV file named data.csv
write_csv(data, "data.csv")
# Data preprocessing and analysis
cleaned_data <- data %>%
filter(!is.na(SDG_Index_Score)) %>%
mutate(Year = as.numeric(Year))
# Create correlation analysis
correlation_data <- cleaned_data %>%
select(SDG_Index_Score, Goal_4_Quality_Education, Goal_1_No_Poverty) %>%
cor(use = "complete.obs")
# Create interactive visualization
plot <- cleaned_data %>%
ggplot(aes(x = Goal_4_Quality_Education, y = SDG_Index_Score)) +
geom_point(aes(color = Region), alpha = 0.7) +
geom_smooth(method = "lm", se = TRUE) +
labs(title = "SDG Index Score vs Quality Education",
x = "Quality Education Score",
y = "SDG Index Score")
